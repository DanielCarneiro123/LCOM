Draws a sprite partially
This function is used when redrawing the background behind moving/cleaned objects, thus avoiding the need to redraw the entire frame buffer per frame.

ydrawy coordinate from which to start painting pixels (relative to the sprite, that is, 0 equates to the parameter passed for "y")
heightHeight of fragment to be drawn
widthWidth of fragment to be drawn
Returns
1 on failure, 0 otherwise
